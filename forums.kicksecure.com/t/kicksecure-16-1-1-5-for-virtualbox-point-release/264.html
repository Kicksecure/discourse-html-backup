What is a Point Release?

Major Changes

  • add fuse to kicksecure-default-applications-cli because fuse is required when running flatpak as user, by appimages. installation of monero-gui from flatpak fails without sudo su - Support - Whonix Forum
  • install xscreensaver by default by adding xscreensaver to kicksecure-shared-host-xfce Kicksecure no screen lock installed
  • add torsocks to kicksecure-recommended-cli useful to download from onions using curl (such as Linux installers)
  • abolish architecture specific syscall whitelist to improve maintainability Sdwdate failed to start in debian 12 bookworm (next release) - #6 by Patrick
  • apparmor fixes
  • vm.mmap_rnd_bits: Fix ppc64le Probably fixes a bunch of other non-x86_64 arches too.
    vm.mmap_rnd_bits: Fix ppc64le · Kicksecure/security-misc@61f6325 · GitHub
    (Thanks to @JeremyRand)
  • derivative-maker:
    • more robust mount, umount, umount_kill.sh
    • work towards systemd-nspawn

Upgrade

Alternatively, in-place release upgrade is possible upgrade using Kicksecure repository.
