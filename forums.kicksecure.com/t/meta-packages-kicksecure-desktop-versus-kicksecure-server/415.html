Meta Packages: Kicksecure Desktop versus Kicksecure Server

Development
Patrick March 18, 2024, 1:23pm 1

GitHub - Kicksecure/kicksecure-meta-packages needs improvements:

  • kicksecure-cli - keep existing package as is
  • kicksecure-cli-server - new package to be added to kicksecure-meta-packages
    • Depends: kicksecure-desktop-firewall
      • includes firewall that by default opens port 22 (SSH’s default port)
    • security-misc-server [no MAC address randomization]
  • kicksecure-cli-desktop - new package to be added to kicksecure-meta-packages
    • Depends: kicksecure-desktop-firewall - new package to be created
      • Comes with all ports closed by default.
    • Depends: security-misc-desktop - new package to be created
      • Might come with MAC address randomization. → ticket
  • kicksecure-xfce - keep existing package as is
  • kicksecure-xfce-server [Will not be created.]
  • kicksecure-xfce-desktop - new package to be created
    • Depends: kicksecure-cli-desktop
