Major Changes

  • grub-live:
    • fix dracut vs initramfs-tools package installation check
  • helper-scripts:
    • add chmod-calc
    • str_replace: Avoid closing and re-opening target file (Thanks to @ArrayBolt3!)
    • str_replace, str_match: Don’t allow massive file reads to block SIGINT termination (Thanks to @ArrayBolt3!)
    • improve str_match
    • improve str_replace
  • msgcollector:
    • minor fix: check if msgcollector-gui systemd user unit is masked and don’t start msgcollector-gui in that case
  • systemcheck:
    • fix systemcheck start from start menu
  • tor-control-panel:
    • fix restart-tor-gui
  • ISO:
    • Fix cosmetic unicode.pf2 not found GRUB error (Thanks to @ArrayBolt3!)
      derivative-maker
    • Enable building CLI Kicksecure ISOs (Thanks to @ArrayBolt3!)
    • Disable debian-installer, needs more testing (Thanks to @ArrayBolt3!)
